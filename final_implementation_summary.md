## Lessons Learned

### Technical

1. **Model Availability**: Claude 3.5 Sonnet not available ‚Üí Used Haiku ‚Üí 90% cost savings
2. **Routing Accuracy**: Generic keywords caused issues ‚Üí Word boundaries + prioritization ‚Üí 100% accuracy
3. **Environment Loading**: Module-level init failed ‚Üí Lazy initialization ‚Üí Works perfectly
4. **Database Setup**: REST API doesn't support DDL ‚Üí Use SQL Editor ‚Üí Proper schema

### Process

1. **Test Early**: Deployment tests caught issues before production
2. **Document Everything**: Comprehensive docs made handoff easy
3. **Iterate Fast**: Quick fixes improved accuracy from 58% ‚Üí 100%
4. **Cost Conscious**: Model selection saved 94% on costs
